# task-manager-kotlin
Task manager based on Devaria 2021 course, done whilst learning Kotlin language.

## Tools/Languages/Frameworks
- Kotlin
- Intellij
- MySQL DB hosted on Microsoft Azure
- Basic Security with JWT and Cors
- Springboot
- Swagger
- Postman

## Features
- [x] Login
- [x] User Signup
- [ ] Add task
- [ ] Get tasks
- [ ] Update tasks
- [ ] Delete task
